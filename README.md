# SYCL examples

Some simple SYCL code.

## N-body Maxwell-Boltzmann distribution

Uses N-body simulation to
approximate [Maxwell-Boltzmann distribution](https://en.wikipedia.org/wiki/Maxwell%E2%80%93Boltzmann_distribution).
